Processus d'importation dans base de données des membres

Mise à jour existants

Les coordonnées (téléphone, courriel) des membres dont le no de HCR est trouvé dans la base de données des membres sont mises à jour avec les données du fichier HCR.

Aucun enregistrement n'est ajouté à la liste des membres.

Ajout nouveaux

Les données HCR qui n'ont aucune correspondance dans la liste des membres (en utilisant le no de HCR pour identifier le membre) sont ajoutés à la liste des membres. Les informations sur les parents ne sont pas utilisées. Les membres ajoutés sont donc tous initialement orphelins.

Orphelins

Permet de trouver les parents d'un joueur dans la liste des membres, s'il y est. Les parents existants dans la liste des membres sont proposés comme parent du joueur sont

  • ceux qui ont le même nom que le parent inscrit dans le dossier HCR
  • ceux qui ont le même nom de famille que dans le dossier HCR, et le même code postal.

Mise à jour parents

Pour chaque parent, affiche côte à côte les coordonnées (téléphone, courriel) de la base de données des membres et celles possiblement mises à jour du fichier HCR. Permet de désigner les informations à transférer de HCR à la liste de mebres les informations jugées les plus à jour.
